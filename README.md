# Source Xeno
Xenophobia plugin for auto kicking players from desired country and city or manually

### Commands
- ``!xeno <COUNTRY_CODE,COUNTRY_NAME,CITY_NAME>``: Kick desired players from specific country or city

### Command Lines
- ``-blockCountries``: ``-blockCountries Chile,Brazil,China``
- ``-blockCities``: ``-blockCountries Santiago,SaoPaulo,Beijing``
